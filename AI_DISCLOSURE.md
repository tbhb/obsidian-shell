# Disclosure statement for AI tool usage

<!-- vale off -->

## Artificial intelligence tool

[Claude](https://claude.ai) by [Anthropic](https://www.anthropic.com), accessed through [Claude Code](https://docs.anthropic.com/en/docs/claude-code), [claude.ai](https://claude.ai), and the [Claude desktop app](https://claude.ai/download) (chat and Claude Cowork) throughout development.

## Methodology

Claude helped design the development toolchain, including linting configuration, pre-commit hooks, prose linting with Vale, and CI/CD pipeline setup. Claude also helped design the release pipeline, including the release-please configuration and the per-platform native-build matrix that ships node-pty binaries as release assets. Human authors made all final tooling decisions.

## Information collection

Claude researched libraries and best practices by reading documentation and fetching web resources during development.

## Data collection method

Claude contributed to the design and development of the obsidian-shell software itself under human direction.

## Execution

Claude drafted code and tests. Human authors made all architecture and design decisions and reviewed all output.

## Interpretation

Claude assisted with debugging, code review, error diagnosis, and suggesting fixes. Human authors evaluated and applied all suggestions.

## Privacy and security

No user data or secrets have been shared with Claude during development. Only source code and public documentation pass through the tool.

## Writing&#8212;Review & Editing

Claude drafted and revised documentation and commit messages. Human authors reviewed and edited all content.

<!-- vale on -->

Human authors take full responsibility for the correctness of this software.

## About this statement

This disclosure adapts the Artificial Intelligence Disclosure (AID) method introduced by Weaver (2024) for use in open source software development.
